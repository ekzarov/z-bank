# Build and Deploy Bank of Z

## Overview
Build the Bank of Z application and deploy the generated artifacts to the target z/OS environment.

## What You Will Learn

- How the build process transforms source code into deployable artifacts
- How deployment packages are generated
- How application components are deployed to runtime environments
- How to validate a successful deployment

## Outcome

A fully deployed Bank of Z environment that is ready for development and testing activities.
