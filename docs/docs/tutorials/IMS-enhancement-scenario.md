# IMS Enhancement Scenario

## Overview

Enhance an existing IMS application component and deploy the updated resources using the Bank of Z development workflow.

## What You Will Learn

- IMS integration patterns
- Updating IMS-related components
- Build and deployment workflows
- Testing and validation procedures

## Outcome

A deployed IMS enhancement that demonstrates IMS application updates, build automation, deployment, and validation.
