# Legacy Analysis Workspace

This directory contains reusable reverse-engineering material copied from the
XPlanner migration project:

- `legacy_user_flows_template.xlsx` - empty workbook template.
- `legacy_user_flows_template_instructions.md` - rules for evidence-based flow
  discovery, SDD mapping, implementation approval, and parity tracking.
- `tools/` - workbook audit utility and its locked Node.js dependencies.

The completed XPlanner workbook and temporary rendering files were deliberately
excluded because they contain project-specific evidence. Bank of Z analysis
must start from the empty template and evidence found under `legacy/`.
